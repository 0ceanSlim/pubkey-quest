🗺️ Development Roadmap

Track the progress of Pubkey Quest from Pre-Alpha to Beta and beyond.

✅ Completed

✓ Nostr Authentication
One MILL sign-in modal for every method — NIP-07 extensions, Amber, remote bunkers (NIP-46), private keys, and brand-new identities. All signing happens in your browser; the server only ever sees your public key.
✓ Deterministic Character Generation
Race, class, background, alignment, ability scores, starting equipment and spells from Nostr npub seed. Cinematic multi-scene introduction with personalized backstories.
✓ Save System (local JSON, per-user directories)
✓ Item Database (200+ D&D 5e items with full properties)
✓ Spell Database (full D&D 5e spell list with components, effects)
✓ Inventory System
Drag-and-drop, context menus, item stacking, ground items, 10 equipment slots, backpack grid.
✓ Container System (backpacks, pouches, nested inventory)
✓ Equipment System
10 gear slots with validation, stat effects from worn items, equip/unequip with backend authority.
✓ Location System
9 cities, 9 environments, districts, and buildings.
✓ NPC System
22 NPCs across 6 cities with daily schedules and time-aware dialogue.
✓ Shop/Merchant System
Buy/sell with charisma price scaling, merchant gold and inventory tracking, restocking.
✓ Vault Storage (per-city secure storage)
✓ Time & Tick System
24-hour day/night cycle with tick-based game loop.
✓ Effects System
Data-driven with 27+ effects. Buffs, debuffs, status conditions, worn item effects. Fully managed via CODEX.
✓ Hunger & Fatigue (accumulation over time, consequences, food/rest mechanics)
✓ Encumbrance/Weight System (weight tracking, overweight/encumbered effects)
✓ Music System (location-based tracks, in-game player)
✓ Spellcasting
Prepare and cast spells in and out of combat, paying mana and material components (or using a focus). Drink potions, throw items, and read spell scrolls mid-fight.
✓ Class Abilities, Conditions & Saves
Class abilities (Rage, Second Wind, Ki…), status conditions (poisoned, prone, frightened…), and saving throws round out every class's turn.
✓ Feats (take a feat instead of ability points at level-up)
✓ Gathering & Lockpicking (mining/fishing nodes and locked chests — tool-gated skill checks with tiered loot)
✓ Installable Web App (add Pubkey Quest to your home screen)
✓ Delta Architecture (surgical DOM updates, backend-authoritative state)
✓ CODEX Tool Suite
Item editor, starting gear editor, systems/effects editor, validation, and database migration.
✓ CI/CD Pipeline (GitHub Actions, auto-deploy test server)
✓ ES6 Module System (Vite build, modern JS architecture)
✓ Turn-Based Combat
Grid combat with initiative, two-phase turns, 0–6 range, full weapon properties, ammo, opportunity attacks, disengage/dash, hold & ready, flee, death saves, loot, and a defeat penalty.
✓ World Travel (traverse environments between towns, live progress, fatigue, arrival discovery)
✓ Points of Interest
Discover landmarks & dungeons while travelling, then explore them as branching node walks — narrative, choices gated by skills/items, skill checks, loot, and combat that resumes the walk on victory.
✓ Encounters (random biome fights scaled to level + hand-authored roadside vignettes)
✓ Quest System
Main / side / class / race quests started by talking to NPCs, objective tracking (slay, fetch, explore, talk, skill-check), stage rewards, an on-screen objective tracker, and rotating daily quests handed out by innkeepers.
✓ Character Progression
Level up from XP, spend ability points, growing HP/mana — a real "get stronger" loop.
✓ Building Rooms (enter inns/taverns, rent a room, sleep, room-scoped NPCs)

🔥 Current Focus: Road to Alpha

The core loop — explore, fight, cast, quest, level up — is in, and combat is complete. These are the last pieces before the alpha release:

⟳ Presentation Pass
NPC dialogue now types out on the scene itself; still to come — sharper scaling and fonts, richer equipment/stat readouts, and a general UI polish sweep.
⟳ Crafting & Minigames
Craft spell scrolls at a workbench, and play little mining, fishing, and lockpicking minigames at the gathering nodes and locked chests.
⟳ Content & Release Pass
Author the remaining quest NPCs, items, and monsters, wire up a guided tutorial, and do the final release engineering.

🎯 Alpha to Beta

Once the presentation pass, crafting & minigames, and content pass land, it's alpha. Road to beta:

☐ Major bug cleanup and optimization
☐ UI polish, theming, redesign
☐ Game interaction hydration (more NPC dialogue, meaningful interactions)
☐ More content: music, art, sound effects, UI effects
☐ CODEX expanded to cover all game data facets
☐ Content polish (improving existing content, no expansion)

🌐 Beta Phase (Nostr Integration)

Beta begins when saves move to Nostr relays. No more local saves.

☐ Nostr Relay Integration (save files as Nostr events)
☐ Save file optimization (must be small for relay storage)
☐ Save structure finalized (no breaking changes after this)
☐ Additional Nostr integrations (badges, in-game notes, long-form articles)
☐ Community Wiki
Note: No character wipes during beta (only on full release)

🚀 Future (Post-Beta)

☐ Cross-Client Play
☐ Multiplayer (party system, trading, PvP)
☐ Leaderboards & Rankings

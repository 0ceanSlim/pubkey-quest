Spell Scroll Crafting System

Design

Designed comprehensive spell scroll crafting system allowing spellcasters to create consumable spell scrolls. System includes intelligence-based success rates (base chance + scaling with INT, capped at 95%), component requirements, gold costs, and class restrictions. Features 30+ scroll recipes across Wizard, Cleric, and Druid spells. Any magic class can use scrolls, but only specific classes can craft them. Success chances improve as player intelligence increases beyond minimum requirements.

Martial Class Abilities System

Design

Designed unique ability systems for non-spellcasting classes (Fighter, Barbarian, Monk, Rogue) to balance against magical classes. Each class gets 6 unique abilities with 4 power tiers that unlock and improve with level progression (1-4, 5-9, 10-14, 15+). Classes have unique resource systems: Fighters use Stamina, Barbarians use Rage, Monks use Ki Points, and Rogues use Cunning Points. Abilities include combat maneuvers, defensive techniques, and special attacks that scale in power as characters level up.
